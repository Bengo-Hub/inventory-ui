# Inventory UI

BengoBox inventory management UI. Catalog, warehouses, adjustments, platform admin.

## Dev

```bash
pnpm install
pnpm dev
```

## Deploy

Deployed at `https://inventory.codevertexitsolutions.com`. CI on push to `main`.

## Push from this folder (own repo)

```bash
# From: inventory-service/inventory-ui
git remote add origin https://github.com/YOUR_ORG/inventory-ui.git
git branch -M main
git push -u origin main
```
